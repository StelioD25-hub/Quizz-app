# Human Behavior Quiz

An interactive quiz application built with vanilla JavaScript.
The app features dynamic question flow, streak mechanics, and an AI-style personalized feedback system.

## Features
- Multiple-choice quiz with instant feedback
- Streak tracking and visual rewards
- Personalized end-screen results
- AI-style roast system based on user performance
- Light and dark mode support

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Netlify
- GitHub

## What I Learned
- Frontend state management
- DOM manipulation and event handling
- Designing feedback-driven user experiences
- Structuring scalable frontend logic
